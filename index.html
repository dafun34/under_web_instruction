Инструкция по использованию

Для начала работы вам необходимо сгенерировать VLESS-ключ.
После генерации вы получите ссылку следующего формата:

vless://UUID@SERVER:PORT?security=none&type=ws&path=/somepath#MyVLESS

Новые пользователи с легкостью смогут сгенерировать тестовый ключ на 7 дней в главном меню для ознакомления с сервисом.
После истечения срока вы сможете продлить доступ на платной основе.

Этот ключ может быть использован в клиенте Hiddify или v2box.

Установка клиента для мобильных устройств

V2Box

  1. Скачайте и установите V2Box:
  2. Google Play

  3. App Store
  4. Откройте приложение.
  5. Перейдите в раздел Configs.
  6. Нажмите + и выберите Import v2ray uri from clipboard.
  7. Передвиньте ползунок Slide to Connect для подключения.

Установка клиента для Windows, macOS и Linux

Hiddify

  1. Скачайте и установите Hiddify Desktop.
  2. Откройте приложение с правами администратора.
  3. Нажмите + в правом верхнем углу приложения.
  4. Нажмите Добавить из буфера обмена.
  5. Перейдите в раздел Параметры конфигурации
  6. В разделе Варианты маршрутизации поменяйте параметр Регион на Другой
  7. В разделе Входящие параметры поменяйте параметр Режим работы на VPN (Экспериментальный)
  8. Вернитесь в раздел Главный нажмите на кнопку подключения.

Важно

Встречаются редкие случаи, когда не получается подключиться с первой попытки.

Выполните повторное подключение.
